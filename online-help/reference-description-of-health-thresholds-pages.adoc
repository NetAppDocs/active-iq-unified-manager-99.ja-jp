= 概要 の健全性しきい値ページ
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
適切な健常性しきい値ページを使用して、アグリゲートとボリュームのグローバル健常性しきい値を設定し、管理対象外の保護関係について遅延警告しきい値と遅延エラーしきい値のグローバルな値を設定できます。
