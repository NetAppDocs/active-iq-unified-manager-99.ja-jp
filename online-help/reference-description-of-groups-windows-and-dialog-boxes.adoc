= 概要 of groupsのウィンドウとダイアログボックス
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
[グループ]ページを使用して、すべてのグループを表示および管理できます。ストレージオブジェクトのグループルールやアクションは、グループルールタブおよびグループアクションタブから設定することもできます。
